#1 ちいかわ方式（左から１個ずつ並べ変える）
for リスト左から最後まで
    if 左隣より小さい
        左が自身より小さくなるまで一個ずつ左に移動
挿入ソートというらしい

#2 ハチワレ方式（最小値を順々にとってくる）
for リスト長さ
    iからnまでの最小値をとる
    最小値をi番目に持ってくる
選択ソートというらしい

#3 うさぎ方式（2分割を繰り返す）
while i<元のリストの長さ
    if i子要素のリストの長さ>1
        左端を基準値にする
        基準値以上のリストと未満のリストに分割
        if 分割できた
            親リストに
        else  分割できない(片方が空っぽ)
            基準値を一個右にずらす
    else:
        次のリストへ(i+=1)
クイックソートというらしい

#4 島二郎方式（ヒープソート）
for ヒープの数 一番下から一番上まで
    ヒープを入れ替え(親ノードより子ノードが大きければ入れ替える)
    # 親ノードのインデックスがiの時、子ノードは2i, 2i+1になる。
    # つまり親ノードのインデックスの最大値(要するに最後のヒープ)はn//2
    if 入れ替えがあったら
        入れ替えた下のヒープを入れ替え
頂点のノードを最大値として取り出す
for　残りのリスト長さ分
    末尾のノードを頂点に持ってくる
    一番上のヒープを入れ替え
    if 入れ替えがあったら
        入れ替えた下のヒープを入れ替え
